CRITERIO GENERAL
Menos es más. La regla es: si una animación no aporta jerarquía, foco o contexto, no va. El sitio debería sentirse vivo pero nunca distraído. Las marcadas como RECOMENDADA son las que conviene usar; las otras quedan disponibles para casos puntuales o para sumar en versiones posteriores.

— Cierre

Mi recomendación final

No todas las animaciones del catálogo deberían entrar a la web. Acá te dejo el set mínimo que recomiendo para V1, y el resto queda como reserva táctica.

SET RECOMENDADO PARA V1 (10 efectos)
Stagger en hero · Section label dash grow · Italic word swap (debajo del hero) · Underline draw (en títulos clave) · Fade-in on scroll (global) · Parallax sutil (hero) · Ken Burns (hero) · Card lift · Magnet link · Índice editorial slide.

Total: 10 efectos coordinados. Suficiente para que la web se sienta viva sin sobrecargar. El resto del catálogo queda disponible para versiones posteriores o secciones específicas.
QUÉ EVITAR
Animaciones muy bouncy o cartoon (transform: scale(1.2) con springs) — rompen el tono editorial.
Cursor followers + parallax + reveals todos juntos — performance se cae y marea.
Loops permanentes en más de 2 elementos por viewport — distrae de leer.
Animaciones disparadas en mobile sin `prefers-reduced-motion` — accesibilidad básica.
